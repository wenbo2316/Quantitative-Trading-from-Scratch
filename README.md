# Quantitative-Trading-from-Scratch
# Introduction
这是一个理工科研究生自学量化交易的记录与教程，出于对危险和不确定性的追逐，以及对财富自由的渴望而开始进行的一项尝试，即便最终失败也是一段宝贵的经验，只有不断地迎接挑战才能快速成长为自己理想的模样，我非常喜欢Chan博士的两句话“One learns much more from mistakes and near-catastrophes than from successes”, "An individual with limited resources and computing power can still challenge powerful industry insiders at their own game". 
# Tutorial
本项目使用的两本教材是Quantitative Trading & Algorithmic Trading by Ernest Chan
# Basic pipeline of setting up a quantitative trading system
1 找到合适的策略
2 做好历史数据回测
3 建立自动交易系统（同时兼顾Business&Technique）
4 管理金钱和风险
# Rough Experience
1 做短线交易只在开盘前10分钟交易，赚多赚少马上收手
2 策略分为两大类：Momentum & 均值回归
3 考虑历史数据单日最高阶最低价，成交量等
4 没有合适的机会不要下场
